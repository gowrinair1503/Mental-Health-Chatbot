# NLP + Reinforcement Learning Based Mental Health Chatbot

## Project Description
This project presents an intelligent mental health chatbot that uses Natural Language Processing (NLP) to provide supportive and adaptive responses to users. The chatbot detects various mental health conditions such as anxiety, depression, stress, panic attacks, overthinking, PTSD, and OCD, and responds with appropriate self-care suggestions.

It also ensures user safety by recommending professional help when necessary.

---

## Objectives
- Develop a chatbot capable of understanding user emotions using NLP  
- Provide personalized self-care suggestions based on detected conditions  
- Implement reinforcement learning to improve response quality  
- Allow users to accept or reject suggestions and receive alternatives  
- Promote mental well-being through guided assistance  

---

## Features
- Detection of multiple mental health conditions:
  - Anxiety  
  - Depression  
  - Stress  
  - Panic Attacks  
  - Overthinking  
  - PTSD  
  - OCD  

- Intelligent response generation using NLP  
- Reinforcement learning-based improvement of suggestions  
- User feedback system:
  - "Yes" → Accept suggestion  
  - "No" → Provide alternative suggestion  

- Randomized self-care suggestions from predefined datasets  
- Recommendation of guided meditation resources  
- Suggestion to seek professional help when required  
- Simple, calm, and user-friendly interface
